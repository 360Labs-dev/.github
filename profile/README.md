### **promptctl** - Prompt Engineering as Code
[![GitHub](https://img.shields.io/badge/GitHub-100rabhkr%2Fpromptctl-blue)](https://github.com/100rabhkr/promptctl)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Stars](https://img.shields.io/github/stars/100rabhkr/promptctl?style=social)](https://github.com/100rabhkr/promptctl/stargazers)

The definitive CLI tool for treating prompts as code with enterprise-grade testing and monitoring.

**Key Features:**
- ğŸ“„ **Prompts as Code** - Define prompts in Markdown with YAML frontmatter
- âœ… **Test-Driven Development** - Assert output quality using exact matches, regex, semantic similarity, or JSON schema
- ğŸ“Š **Bulk Evaluation** - Run hundreds of test cases in seconds with pass/fail rates and latency metrics
- âš–ï¸ **A/B Testing** - Scientific prompt comparison with statistical confidence intervals
- ğŸ“ˆ **Local Dashboard** - Visualize run history, inspect traces, and analyze costs

### **@360labs/live-transcribe** - Real-time Speech Transcription Library
[![npm](https://img.shields.io/badge/npm-%40360labs%2Flive--transcribe-red)](https://www.npmjs.com/package/@360labs/live-transcribe)
[![Downloads](https://img.shields.io/npm/dm/@360labs/live-transcribe)](https://www.npmjs.com/package/@360labs/live-transcribe)

A comprehensive TypeScript library for live speech transcription with support for multiple providers and real-time capabilities.

**Key Features:**
- ğŸ¤ **Multi-Provider Support** - Web Speech API, Deepgram, AssemblyAI
- âš¡ **Real-time Transcription** - Low-latency streaming transcription
- ğŸ”„ **Provider Switching** - Seamlessly switch between providers
- ğŸ¯ **TypeScript Native** - Full type safety and IntelliSense support
- ğŸª **React Hooks** - Easy integration with React applications
- ğŸ”Œ **Extensible Architecture** - Add custom providers easily

**Installation:**
```bash
npm install @360labs/live-transcribe
# or
yarn add @360labs/live-transcribe
# or
pnpm add @360labs/live-transcribe
```

**Quick Example:**
```typescript
import { LiveTranscribe } from '@360labs/live-transcribe';

const transcriber = new LiveTranscribe({
  provider: 'deepgram',
  apiKey: 'your-api-key',
  language: 'en-US'
});

transcriber.on('transcript', (text) => {
  console.log('Transcription:', text);
});

await transcriber.start();
```

**Use Cases:**
- Voice-enabled applications
- Live meeting transcription
- Customer support call analysis
- Accessibility features
- Voice commands and control
